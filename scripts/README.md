# Scripts – AWS Security Best Practices

This folder contains **PowerShell** and **Python** scripts for AWS security best practices, aligned with the [AWS Security Training Curriculum](https://github.com/supermaj09/AWS_WellSpan_Training) (WellSpan).

| Folder | Description |
|--------|-------------|
| [powershell/](powershell/) | AWS Tools for PowerShell – IAM audit, S3 report, monitoring status, root account check |
| [python/](python/) | boto3 – IAM audit, S3 report, monitoring status, root account check |

All scripts are **read-only** by default (reports and status checks). Use in a non-production account first and follow your organization’s policies.

- **PowerShell:** See [powershell/README.md](powershell/README.md) for prerequisites and usage.
- **Python:** See [python/README.md](python/README.md) and `pip install -r python/requirements.txt`.
