# Quick Start Guide - AWS Security Training

## üöÄ Get Started in 5 Minutes

### Step 1: Set Up Your AWS Account (5 minutes)

1. **Create or Access AWS Account**
   - If you don't have an AWS account, create one at [aws.amazon.com](https://aws.amazon.com)
   - Use AWS Free Tier for practice (12 months free)
   - Enable MFA on your root account immediately

2. **Set Up AWS CLI** (Optional but Recommended)
   ```bash
   # Install AWS CLI
   aws --version
   
   # Configure credentials
   aws configure
   ```

3. **Access AWS Console**
   - Log in to [console.aws.amazon.com](https://console.aws.amazon.com)
   - Familiarize yourself with the interface

### Step 2: Review the Curriculum (10 minutes)

1. **Read the Overview**
   - Review [CURRICULUM_OVERVIEW.md](CURRICULUM_OVERVIEW.md)
   - Choose your learning path (Beginner/Intermediate/Advanced)

2. **Understand the Structure**
   - 9 comprehensive modules
   - Hands-on labs in each module
   - Capstone project at the end

### Step 3: Start Learning (Begin Now!)

1. **Begin with Module 1**
   - Open [README.md](README.md)
   - Navigate to Module 1: AWS Security Fundamentals
   - Follow the learning objectives

2. **Complete Hands-On Activities**
   - Each module includes practical exercises
   - Use AWS Free Tier for practice
   - Document your progress

### Step 4: Track Your Progress

### AWS Free Tier Limits
- Be aware of Free Tier limits
- Monitor your usage to avoid charges
- Set up billing alerts

### Security Best Practices
- Never share AWS credentials
- Use IAM users, not root account
- Enable MFA on all accounts
- Follow least privilege principle

### Cost Management
- Set up billing alerts
- Review AWS Cost Explorer regularly
- Clean up resources after labs
- Use AWS Budgets for cost tracking
