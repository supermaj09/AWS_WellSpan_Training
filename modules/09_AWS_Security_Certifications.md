## AWS Certified Security - Specialty

### Exam Overview

- **Exam Format:**
  - 65 questions
  - 170 minutes (2 hours 50 minutes)
  - Multiple choice and multiple response questions
  - Passing score: 750/1000 (75%)

- **Cost:** $300 USD

- **Validity:** 3 years

- **Prerequisites:**
  - AWS Certified Cloud Practitioner or Associate-level certification recommended
  - 5+ years of IT security experience
  - 2+ years of hands-on AWS security experience

### Exam Domains

#### Domain 1: Incident Response (12%)
- **Topics:**
  - Incident response procedures
  - Forensic analysis
  - Containment and remediation
  - Post-incident activities

#### Domain 2: Logging and Monitoring (20%)
- **Topics:**
  - CloudTrail configuration and analysis
  - CloudWatch monitoring
  - AWS Config
  - GuardDuty
  - Security Hub
  - VPC Flow Logs

#### Domain 3: Infrastructure Security (26%)
- **Topics:**
  - VPC security
  - Security groups and NACLs
  - WAF
  - Shield
  - Network Firewall
  - EC2 security
  - Container security
  - Serverless security

#### Domain 4: Identity and Access Management (20%)
- **Topics:**
  - IAM users, groups, roles, policies
  - MFA
  - Identity federation
  - IAM Access Analyzer
  - Permissions boundaries
  - Service Control Policies

#### Domain 5: Data Protection (22%)
- **Topics:**
  - Encryption at rest and in transit
  - KMS
  - Secrets Manager
  - S3 security
  - Database security
  - Data classification
  - Backup and recovery

### Exam-Taking Strategies

1. **Time Management**
   - Allocate time per question (~2.6 minutes per question)
   - Flag difficult questions
   - Review flagged questions at end

2. **Question Analysis**
   - Read questions carefully
   - Identify key requirements
   - Eliminate obviously wrong answers

3. **Answer Selection**
   - Choose best answer (not just correct)
   - Consider AWS best practices
   - Think about real-world scenarios

4. **Practice**
   - Take multiple practice exams
   - Review explanations
   - Understand why answers are correct/incorrect
